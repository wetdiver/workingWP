== Description ==

Niwas Resort Hotel is an attention grabbing, stylish, inviting and modern hotel and resort WordPress theme for luxury hotel, resort, lodge, home stay, motel, guest house, holiday home, inn, dormitory, cottage, accommodation, farmhouse, service apartment, rooms, cuisine and other such hospitality business. Apart from that it can also be used by restaurants, eateries, food joints, cafes and bakeries. It can very well serve bloggers, travel adventures, travel critic and hotel inspector or reviewer. It has a manageable interface of backend which makes designing website a super easy task. It loads fast and gives a good rank to website in search engine results. This resort and hotel WordPress theme is fully responsive and loads on all browsers smoothly. It can be translated into several other languages and you can write content from left to right as well. It facilitates easy customization through theme customizer wherein you can change the colour scheme, background, font, logo, header, footer and menu styles and slider settings. The theme supports gallery where you can show alluring hotel and resort images; testimonial section to get customer feedback and subscriberâ€™s form to connect with customers. The Bootstrap framework strengthens its base. It is a complete theme for all range of hotels and resorts.

For Homepage Setup:
====================

1. Go to Dashboard >> Pages >> Add New, Create a new page and name it as "Home" then select the template "Custom Home Page" and publish it.

2. Go to Dashboard >> Settings >> Reading, Select the option of Static Page, now select the page you created to be the homepage.

For Topbar Secction:
======================

1. Go to Customizer >> Theme Settings >> Top Bar, add welcome text,  email address, phone number and publish it.

2. Go to Customizer >> Theme Settings >> Social Icons, add social icon urls.

For Slider:
============

1. Go to Dashboard >> Pages >> Add New, Create a new Page and name it, add the content, featured image and publish the page. Repeat the step for other slider pages.

2. Go to Customizer >> Theme Settings >> Slider Settings, Click the checkbox of Show/Hide Slider and select the pages of slider that you have created.

For Facilities Section:
=======================

1. Go to Dashboard >> Posts >> Category, add new category for facility section, name it and publish it.

2. Go to Dashboard >> Posts >> Add New, Create a new Post and name it, set it's featured image (image size 300px x 300px), add the content, assign the facility category and publish the page. Repeat the step for other facility posts.

3. Go to Customizer >> Theme Settings >> Our Facilities, add section title, section subtitle and select the category that you have created and publish it.

For Footer Section:
====================

1. Go to Dashboard >> Appearance >> Widgets, here you can add widgets to the footer widgets areas, which can be seen in footer.

For Copyright Text:
====================

1. Go to Customizer >> Theme Settings >> Footer Text, add footer text here.
